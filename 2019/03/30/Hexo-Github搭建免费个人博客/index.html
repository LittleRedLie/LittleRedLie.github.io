一、本地环境搭建

1. 安装Git

  • 官网下载地址（下载速度极慢）: https://git-scm.com/downloads
  • 阿里云镜像（推荐）： https://npm.taobao.org/mirrors/git-for-windows/
下载完后，默认安装就行（一直点next）

2. 安装node.js

  • 官网下载地址：https://nodejs.org/en/download/

在命令行依次输入：npm install -g hexo-cli
